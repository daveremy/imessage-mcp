# Changelog

## 0.1.0

Initial release.

- **im_status**: Check database access, contacts, chat/message counts
- **im_chats**: List recent chats with resolved contact names
- **im_messages**: Read messages with pagination, reactions, attachments, edited/unsent support
- **im_search**: Full-text search across message history
- **im_send**: Send messages to existing chats via Messages.app
- **im_participants**: List chat participants with contact resolution
- Contact name resolution from Apple Contacts (AddressBook)
- Binary typedstream parser for attributedBody (NSArchiver format)
- CLI mode for direct terminal usage
- MCP server mode for Claude Code integration
- Claude Code plugin manifest and companion skill
